# SEO FARM — Инструкция по установке

## Что это
Бот автоматически создаёт группы ВКонтакте под SEO-запросы.  
Управление — через Telegram. Данные — в Google Sheets.

---

## Шаг 1 — Подготовь Google Sheets

1. Создай новую таблицу на [sheets.google.com](https://sheets.google.com)
2. Создай три листа (вкладки внизу):
   - `Настройки`
   - `Ключи`
   - `Результаты`
3. Скопируй ID таблицы из адресной строки браузера:  
   `docs.google.com/spreadsheets/d/`**ВОТ_ЭТО**`/edit`

---

## Шаг 2 — Получи ключ Google (credentials.json)

1. Открой [console.cloud.google.com](https://console.cloud.google.com)
2. Создай новый проект → любое название → **Создать**
3. Слева: **API и сервисы** → **Библиотека** → найди `Google Sheets API` → **Включить**
4. Слева: **API и сервисы** → **Учётные данные** → **Создать учётные данные** → **Сервисный аккаунт**
5. Введи любое название → **Готово**
6. Нажми на созданный аккаунт → вкладка **Ключи** → **Добавить ключ** → **JSON** → скачается файл
7. Открой скачанный файл в блокноте → скопируй **всё содержимое** (понадобится в шаге 4)
8. В том же файле найди строку `"client_email"` → скопируй email
9. Открой Google таблицу → **Настройки доступа** → вставь email → дай права **Редактора**

---

## Шаг 3 — Создай Telegram бота

1. Напиши [@BotFather](https://t.me/BotFather) → `/newbot`
2. Придумай имя и username боту
3. Скопируй токен — выглядит как `1234567890:AAF...`
4. Узнай свой chat_id: напиши `/start` боту [@userinfobot](https://t.me/userinfobot) → скопируй ID

---

## Шаг 4 — Получи токены

**VK токен:**
1. Открой [vk.com](https://vk.com) → авторизуйся под нужным аккаунтом
2. Открой в браузере эту ссылку:
```
https://oauth.vk.com/authorize?client_id=2685278&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=groups,wall,photos,video,offline&response_type=token&v=5.131
```
3. Нажми **Разрешить**
4. В адресной строке найди `access_token=` → скопируй текст до `&expires_in`

**Groq API ключ (бесплатно):**
1. Зарегистрируйся на [console.groq.com](https://console.groq.com)
2. Создай API Key → скопируй

---

## Шаг 5 — Задеплой на Railway

1. Зарегистрируйся на [railway.app](https://railway.app)
2. **New Project** → **Deploy from GitHub repo**
3. Выбери репозиторий с файлами `main.py`, `requirements.txt`, `Procfile`
4. Нажми на проект → вкладка **Variables** → добавь переменные:

| Название | Значение |
|---|---|
| `TG_TOKEN` | токен бота от BotFather |
| `TG_CHAT_ID` | твой chat_id |
| `VK_TOKEN` | токен VK аккаунта |
| `GROQ_API_KEY` | ключ от Groq |
| `SPREADSHEET_ID` | ID Google таблицы |
| `GOOGLE_CREDENTIALS_JSON` | содержимое credentials.json (весь JSON одной строкой) |
| `SITE_URL` | https://сайт-клиента.ru |
| `MAX_PER_RUN` | 4 |
| `PAUSE_MIN` | 120 |
| `PAUSE_MAX` | 180 |

5. Railway автоматически запустит бота

| частная школа тюмень | Тюмень | | https://ссылка.jpg | https://ссылка.jpg | https://ссылка.jpg | https://ссылка.jpg |

> Фото — прямые ссылки на JPG/PNG. Можно хранить в Google Drive:

> заменить `/view` на `/uc?export=download`

---

## Шаг 7 — Запуск

1. Найди своего бота в Telegram → напиши `/start`
2. Нажми кнопку **▶️ Запустить агента**
3. Агент создаст до 4 групп, потом остановится
4. Запускай каждый день — будет добавлять по 4 группы

---

## Кнопки бота

| Кнопка | Что делает |
|---|---|
| ▶️ Запустить агента | Начинает создание групп |
| ⏹ Остановить | Останавливает после текущей группы |
| 📊 Статус | Показывает работает ли агент |
| 📋 Логи | Описание где смотреть логи |
| ❓ Помощь | Краткая инструкция |

---

## При капче

Если VK запросит капчу — бот пришлёт картинку и два варианта:
- Напиши `/captcha ТЕКСТ` — агент продолжит
- Напиши `/tomorrow` — агент остановится, продолжишь завтра
